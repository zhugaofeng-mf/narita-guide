中日双语 / 中日二ヶ国語ガイド

路线① Access特快（快速推荐）
成田空港 → 押上 → 住吉 → 西大島
全程约70-80分钟｜费用约1500日元
换乘2次，全部站内换乘，不用出站
🔋 第0步：给IC卡充值
0给IC卡充值 到达成田机场后，先在到达大厅找到京成线的售票机，给手机里的IC卡（Suica/PASMO）充值2000日元以上。 到着ロビーで「京成線」の券売機を探して、ICカード（Suica/PASMO）に2000円以上チャージしてください。
充值机上有中文界面，按"チャージ"（充值）按钮即可。
券売機には中国語表示あり。「チャージ」ボタンを押す。
🚃 出发！
1成田空港 → 押上 从到达大厅跟着「京成線 / Keisei」的标识走到地下B1F。
进改札后，去1号站台（アクセス特急专用）。
坐「アクセス特急」（Access特快），往押上/羽田空港方向。
坐到押上站下车。约60分钟。 到着ロビーから「京成線」の案内に従いB1Fへ。
改札を入って1番ホームで「アクセス特急」押上・羽田空港方面に乗車。
押上駅で下車。約60分。
绝对不要坐「スカイライナー」！那个需要额外买特急券，很贵！
スカイライナーは別料金です！アクセス特急に乗ってください！
进改札后注意：不要通过「中间改札口」！直接往1号站台走。
改札を入ったら「中間改札口」を通らず、直接1番ホームへ。

2押上 → 住吉（换乘半蔵門線） 在押上站下车后，不要出改札！
跟着「半蔵門線 Hanzōmon Line」的换乘标识走。
从B1F下到B2F出京成改札，然后往半蔵門線方向走，到B3F站台。
坐半蔵門線往渋谷方向，坐1站到住吉。 押上駅で降りたら改札を出ずに「半蔵門線」乗り換え案内に従う。
B1F→B2F（京成改札出る）→半蔵門線方向→B3Fホーム。

3住吉 → 西大島 到住吉站后，跟着「都営新宿線 Shinjuku Line」标识换乘。
同一层走过去就好，很近。
坐都営新宿線往本八幡方向，坐1站到西大島！ 住吉駅で「都営新宿線」案内に従い乗り換え。同じフロア内。
都営新宿線 本八幡方面に乗車、1駅で西大島！

路线② 京成本線（便宜）
成田空港 → 京成八幡 → 本八幡 → 西大島
全程约100-110分钟｜费用约1300日元
换乘1次，但需要出站步行约5分钟
🔋 第0步：给IC卡充值
0给IC卡充值 到达成田机场后，先在到达大厅找到京成线的售票机，给手机里的IC卡（Suica/PASMO）充值2000日元以上。 到着ロビーで「京成線」の券売機を探して、ICカードに2000円以上チャージしてください。
充值机上有中文界面，按"チャージ"按钮即可。
🚃 出发！
1成田空港 → 京成八幡 从到达大厅跟着「京成線 / Keisei」标识走到地下。
进改札后，需要通过「中间改札口」，到达京成本线专用站台。
去2号或3号站台，坐「快速特急」或「特急」，往京成上野/押上方向。
坐到京成八幡站下车。约80分钟。 「京成線」の案内に従い地下へ。改札を入り、「中間改札口」を通って京成本線ホームへ。
2・3番ホームで「快速特急」または「特急」京成上野・押上方面に乗車。
京成八幡駅で下車。約80分。
注意！京成本线和Access线的站台是分开的！必须通过「中间改札口」才能到2/3号站台。
IC卡刷两次改札是正常的，不要紧张。
ICカードで2回タッチは正常です。

⬇️
2京成八幡 → 步行 → 本八幡 京成八幡是高架站（楼上）。下车后坐扶梯到1F，出改札。
出站后往左走，沿着大马路（国道14号）直走约5分钟，就能看到本八幡站的入口（地下站）。
进入都営新宿線 本八幡站。 京成八幡は高架駅です。下車後エスカレーターで1Fへ、改札を出ます。
出たら左に曲がり、国道14号沿いに約5分歩くと本八幡駅（都営新宿線・地下）の入口が見えます。
带大行李箱的话走室外要注意，约5-7分钟路程。
スーツケースがある場合、歩道にご注意ください。

3本八幡 → 西大島 进入本八幡站（地下站），刷IC卡进改札。
坐都営新宿線往新宿方向，坐约15分钟（6站）到西大島！ 本八幡駅に入り、ICカードで改札を通ります。
都営新宿線 新宿方面に乗車、約15分（6駅）で西大島！
